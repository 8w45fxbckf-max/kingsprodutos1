Kings Produtos — Modelo Moderno Minimalista
==========================================

Conteúdo do ZIP:
- index.html        -> Página inicial (com destaques)
- products.html     -> Catálogo de produtos (com busca)
- contact.html      -> Formulário de contato (salva mensagens em localStorage)
- login.html        -> Entrar / Criar conta (demo em localStorage)
- cart.html         -> Página do carrinho (adicionar, remover, finalizar - demo)
- css/styles.css    -> Estilos modernos minimalistas
- js/app.js         -> Lógica de produtos, carrinho e autenticação demo

Como usar:
1. Extraia o ZIP em uma pasta.
2. Abra `index.html` no navegador.
3. Este é um protótipo estático. Para pagamentos e autenticação real, integre um back-end
   e um gateway de pagamento (Mercado Pago, PagSeguro, Stripe etc.).

Personalização:
- Edite `js/app.js` para atualizar produtos (título, preço, imagem, descrições).
- Troque imagens por URLs reais ou faça upload para uma pasta `assets/` e atualize os caminhos.
- Posso gerar uma versão com arquivos separados prontos para deploy (Netlify/Vercel) ou integrar
  com um back-end
